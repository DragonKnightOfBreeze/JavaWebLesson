裁决之龙

 《裁决之龙》是游戏王卡片中的一张效果怪兽卡，隶属光道卡组。拥有清场的效果。该卡不能通常召唤，自己墓地的「光道」怪兽是4种类以上的场合才能特殊召唤。

调整信息

[09/09/09] [1]

●这张卡不能通常召唤。自己墓地名字带有「光道/ライトロード」的怪兽卡有4种类以上存在的场合才能特殊召唤。
◇召唤条件（非效果，不进入连锁）。
◇有没有完成正规召唤程序都不能用其他卡的效果特殊召唤。

●可以支付1000基本分，把除这张卡外场上的卡全部破坏。
◇起动效果（进入连锁）。
◇支付1000基本分是COST。
◇效果处理时这张卡里侧表示存在的场合仍只是除这张卡外场上的卡全部破坏。

●这张卡在自己场上表侧表示存在的场合，每次自己的结束流程，从卡组上面把4张卡送去墓地。
◇诱发效果（进入连锁）。
◇强制发动。
◇卡数不足的场合仍发动。
◇发动被「光与暗之龙/光と闇の竜」无效的同一个结束阶段会重复发动。
◇效果处理时这张卡不在自己场上表侧表示存在的场合，效果不发。
◇效果处理时卡数不足的场合，全送去墓地。
